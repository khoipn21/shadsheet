# useAutoFill

Handles the fill handle drag for extending values.

## Usage

```tsx
import { useAutoFill } from "@shadsheet/ui";

const { fillHandlePosition, handleFillStart, handleFillEnd } = useAutoFill({
  activeCell,
  selectionRange,
  visibleColumnIds,
  totalRowCount,
  hf,
  incrementRenderTrigger,
});
```
