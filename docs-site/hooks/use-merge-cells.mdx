# useMergeCells

Handles merging and unmerging cells.

## Usage

```tsx
import { useMergeCells } from "@shadsheet/ui";

const {
  toggleMerge,
  unmerge,
  canMergeSelection,
  isSelectionExactlyMerged,
  getMergedCellAt,
} = useMergeCells({
  visibleColumnIds,
  leftPinnedCount,
  centerCount,
  onClearCellValue,
});
```
